Jak stworzyć stronę internetową?

Aby stworzyć stronę internetową należy:
  1. Stworzyć pusty plik index.html
  2. Stworzyć pusty plik styles.css
  3. Wpisać w pliku index.html następującą treść: 
    
<!DOCTYPE html>
      <html>
      <head lang="en">
          <meta charset="UTF-8">
          <title></title>
      </head>
      <body>

      </body>
 </html>
  4. Aby zrobić ciemne tło na stronie należy w pliku styles.css wpisać: 
             
body {
    background-color: rgba(8, 8, 8, 0.82);
}
